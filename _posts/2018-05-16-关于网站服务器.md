## http-server:
* 一个命令行http服务器

* 全局安装:
* 
		npm install http-server -g
* 用法:
    http-server [path] [options]
    
    [path]默认为 ./public目录，如不存在则 ./目录
    
    启动后可用http://localhost: 8080来查看服务器

* 可用选项：
    -p 端口 （默认为8080）
    -a 地址  （默认0.0.00）
    -d 显示目录列表 （默认为‘true’）
    -S或--ssl 启用https
